# QuiltCorgi Keyboard Shortcuts Reference

## Canvas Navigation

| Shortcut | Action |
|----------|--------|
| **Space + Drag** | Pan the canvas |
| **Ctrl + Scroll** | Zoom in/out |
| **Ctrl + 0** | Reset zoom to 100% |

## Editing

| Shortcut | Action |
|----------|--------|
| **Ctrl + Z** | Undo last action |
| **Ctrl + Shift + Z** | Redo undone action |
| **Ctrl + Y** | Redo undone action (alternative) |
| **Ctrl + C** | Copy selected object(s) |
| **Ctrl + V** | Paste copied object(s) |
| **Ctrl + X** | Cut selected object(s) |
| **Delete** | Remove selected object(s) |
| **Backspace** | Remove selected object(s) |
| **Ctrl + A** | Select all objects |
| **Ctrl + D** | Duplicate selected object(s) |

## Tool Selection

| Shortcut | Tool |
|----------|------|
| **V** | Select tool |
| **R** | Rectangle tool |
| **T** | Triangle tool |
| **P** | Polygon tool |
| **L** | Line tool |
| **C** | Curve tool |
| **X** | Text tool |
| **Esc** | Cancel current tool/selection |

## Panel Toggles

| Shortcut | Action |
|----------|--------|
| **B** | Toggle Blocks panel |
| **F** | Toggle Fabrics panel |
| **I** | Toggle Puzzle view (Piece Inspector) |

## Project Management

| Shortcut | Action |
|----------|--------|
| **Ctrl + S** | Save project |
| **Ctrl + E** | Export image |

---

## Notes

- **Ctrl** on Windows/Linux = **Cmd** on Mac
- Shortcuts are disabled when typing in text inputs
- Some shortcuts require an active selection (e.g., Ctrl+C, Delete)
- Tool shortcuts (V, R, T, etc.) work without modifier keys

## Accessibility

All keyboard shortcuts are designed to work with screen readers and assistive technologies. Focus indicators are visible when navigating with Tab/Shift+Tab.

For help, press **?** or click the Help button in the studio.
