# Using the Block Library

QuiltCorgi ships with 659+ quilt blocks organized across 20+ categories. This tutorial shows you how to find, preview, and place blocks in your quilt design.

## Opening the Block Library

Click the **Block Library** icon in the tool rail (it looks like a grid of small squares) or press `B` to open the library panel. The panel slides open on the right side of the canvas, giving you a scrollable grid of block thumbnails.

## Browsing by Category

Blocks are organized into intuitive categories:

| Category | Examples | Count |
|----------|----------|-------|
| Stars | Ohio Star, Lone Star, Hunter's Star | 80+ |
| Log Cabins | Traditional, Courthouse Steps, Pineapple | 40+ |
| Pinwheels | Basic Pinwheel, Double Pinwheel, Windmill | 30+ |
| Pictorial | Animals, Houses, Trees, Flowers | 60+ |
| Holiday | Christmas, Halloween, Patriotic | 50+ |
| Art Deco | Geometric, Fan, Clamshell | 35+ |
| Celtic | Knot, Braid, Interlace | 25+ |
| Traditional | Nine-Patch, Flying Geese, Bear's Paw | 100+ |

Click a category tab to filter the grid. The number badge on each tab shows how many blocks are in that category.

## Searching for Blocks

The search bar at the top of the library panel lets you find blocks by name. Type a few characters and the results update instantly. Try searching for:

- **"star"** to see all star variations
- **"cabin"** for log cabin blocks
- **"bear"** to find Bear's Paw and related designs

> **Tip:** Search is fuzzy — you do not need exact names. Typing "pinwhl" will still find Pinwheel blocks.

## Previewing a Block

Click any block thumbnail to open a larger preview. The preview shows:

- The block at full resolution with default coloring
- The block name and category
- A wireframe overlay showing seam lines
- The number of unique patches and templates

This helps you evaluate complexity before adding the block to your design.

## Adding Blocks to Your Quilt

There are three ways to place a block:

1. **Drag and drop** — Click and hold a block thumbnail, then drag it onto any cell in your quilt grid
2. **Double-click** — Select a cell on the canvas first, then double-click a block to fill that cell
3. **Fill all** — Right-click a block and choose "Fill All Empty Cells" to populate your entire grid with that block

## Replacing and Removing Blocks

To replace a block, simply drag a new block onto an occupied cell. The old block is replaced while preserving any color or fabric assignments where patches overlap.

To remove a block, select the cell and press `Delete` or right-click and choose "Clear Cell."

## Block Coloring in the Library

Blocks in the library use default coloring for clarity, but you can recolor them after placement. Select a placed block, then use the context panel to assign your own colors or fabrics to each patch.

> **See also:** [Drawing Custom Blocks with EasyDraw](/tutorials/custom-blocks-easydraw) if you want to create a block that is not in the library.

## Custom Blocks in the Library

Any block you create with [EasyDraw](/tutorials/custom-blocks-easydraw) or the [Applique tool](/tutorials/applique-designs) is automatically saved to a "My Blocks" category at the top of the library. These blocks work exactly like built-in blocks — you can drag, preview, and reuse them across projects.
