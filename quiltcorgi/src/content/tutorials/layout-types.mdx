# Layout Types: Grid, On-Point, Medallion & Lone Star

QuiltCorgi supports six layout types, each producing a distinct look from the same set of blocks. This tutorial explains each layout and when to use it.

## Switching Layouts

Open **Layout Settings** from the context panel (gear icon) while on the Quilt worktable. The layout selector shows all available options with thumbnail previews. Click a layout type to apply it — your blocks are rearranged instantly.

> **Important:** Switching layouts preserves your block assignments and colors. However, some layouts use different numbers of cells, so blocks may be added or removed from the edges.

## Grid Layout

The grid is the most common and simplest quilt layout. Blocks are arranged in straight rows and columns.

### Settings

- **Rows** and **Columns** — Set the grid dimensions
- **Block size** — Set the finished size of each block (e.g., 12 inches)
- **Alternating blocks** — Optionally fill every other cell with a second block

### Best For

- Traditional quilt designs
- Sampler quilts with different blocks in each cell
- Photo quilts and pixel-style designs
- Beginners learning to design

## Sashing Layout

Sashing adds strips of fabric between blocks, creating a window-frame effect that separates and highlights each block.

### Settings

- **Sashing width** — The strip width between blocks (e.g., 2 inches)
- **Sashing color/fabric** — Assign a color or fabric to the strips
- **Cornerstones** — Optional small squares where sashing strips intersect
- **Cornerstone color** — A contrasting color for cornerstones adds visual interest

### Best For

- Showcasing individual blocks with breathing room
- Sampler quilts where each block is unique
- Adding structure to busy, high-contrast blocks

## On-Point Layout

On-Point rotates the entire grid 45 degrees so blocks sit on their corners like diamonds. Setting triangles fill the edges to create a straight border.

### Settings

- **Rows** and **Columns** — The number of on-point blocks
- **Setting triangles** — Automatically generated half-square and quarter-square triangles at the edges
- **Side triangle fabric** — Choose the fabric for edge triangles (often matches the background or border)

### Best For

- Adding movement and energy to a design
- Traditional quilts like Barn Raising or Trip Around the World
- Making square blocks feel more dynamic

> **Tip:** On-Point layouts use more fabric for the setting triangles. QuiltCorgi calculates the additional yardage and includes it in your [cutting chart](/tutorials/exporting).

## Medallion Layout

Medallion quilts radiate outward from a central focal point. Each round adds a ring of blocks or borders around the center.

### Settings

- **Center block** — A single feature block (can be larger than the surrounding blocks)
- **Rounds** — Add concentric rings of blocks or borders
- **Round type** — Each round can be blocks, pieced borders, or solid borders
- **Round width** — Set the width of each round independently
- **Border patterns** — For pieced border rounds, choose from flying geese, half-square triangles, or other patterns

### Best For

- Showcasing a special center block
- Traditional medallion and Baltimore Album quilts
- Building complexity outward from a simple center

### Adding Rounds

1. Click **Add Round** in the Layout Settings panel
2. Choose the round type (blocks, pieced border, or solid border)
3. Configure the round width and block/pattern selection
4. Repeat to add more rounds

Each round is independently configurable, letting you mix solid borders with pieced borders and block rounds.

## Lone Star Layout

The Lone Star is an eight-pointed star that radiates from the center of the quilt. Each diamond-shaped ring uses different colors to create the starburst effect.

### Settings

- **Number of rings** — How many concentric diamond rings (typically 4-8)
- **Ring colors** — Assign a color or fabric to each ring
- **Background** — The fabric filling the spaces between star points
- **Corner treatment** — Fill corners with blocks, setting triangles, or applique

### Best For

- Dramatic, show-stopping quilts
- Exploring color gradients and value progressions
- Advanced quilters wanting a challenging, rewarding project

> **Tip:** Use the [Generators](/tutorials/generators) tool with Lone Star layouts for automatic color gradient suggestions across rings.

## Borders (All Layouts)

All layout types support borders. Access border settings from the **Borders** section in Layout Settings:

- **Solid borders** — Simple fabric strips at a specified width
- **Pieced borders** — Flying geese, half-square triangles, or custom patterns
- **Multiple borders** — Stack several borders for a framed effect
- **Corner treatment** — Mitered corners, cornerstones, or pattern-matched corners
