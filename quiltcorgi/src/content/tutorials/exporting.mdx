# Exporting: PDF Templates, FPP & Cutting Charts

Once your quilt design is complete, QuiltCorgi helps you turn it into real-world sewing patterns. This tutorial covers all the export options available from the **Print worktable**.

## Switching to the Print Worktable

Click the **Print** tab in the top bar's segmented control. The Print worktable shows your quilt design with printing annotations — seam allowances, template labels, and registration marks.

## PDF Template Export

The primary export format is a multi-page PDF with actual-size templates for every unique patch in your quilt.

### Setting Up Your Export

1. Click **Export PDF** in the Print worktable toolbar
2. Configure your settings:
   - **Paper size** — Letter, A4, Legal, or custom
   - **Seam allowance** — 1/4 inch (standard) or custom value
   - **Template style** — Cut line only, cut + sew line, or sew line only
   - **Grain line arrows** — Show recommended grain line direction on each template
   - **Label format** — Template ID, block name, or both

### What the PDF Includes

Your exported PDF contains:

1. **Cover page** — Quilt name, finished dimensions, block count, color/fabric summary
2. **Assembly diagram** — Full-quilt overview showing block placement with labels
3. **Template pages** — One page per unique template shape at actual size with seam allowance
4. **Cutting instructions** — How many of each template to cut, organized by fabric/color

> **Tip:** Templates are deduplicated — if the same shape appears in multiple blocks, it prints once with the total count needed.

## Foundation Paper Piecing (FPP) Export

FPP templates let you sew complex blocks with precision by stitching directly onto paper patterns.

### Generating FPP Templates

1. Click **FPP Export** in the Print worktable
2. Select which blocks to generate FPP patterns for
3. Configure options:
   - **Numbering** — Automatic sewing order numbering on each section
   - **Trim lines** — Include trim lines for each section
   - **Mirror** — FPP patterns are automatically mirrored for sewing (the printed side faces away from you)

### Understanding FPP Output

Each FPP template page shows:

- Numbered sewing sections in the correct piecing order
- Seam lines (solid) and trim lines (dashed)
- A color reference showing which fabric goes in each section
- Registration marks for multi-page blocks that span multiple sheets

> **Note:** Not all blocks are suitable for FPP. QuiltCorgi analyzes your block's seam line structure and warns you if a block cannot be foundation pieced in a single pass. See [Drawing Custom Blocks with EasyDraw](/tutorials/custom-blocks-easydraw) for designing FPP-friendly blocks.

## Rotary Cutting Charts

For quilters who prefer rotary cutting over templates, QuiltCorgi generates strip-cutting charts.

### How Cutting Charts Work

1. Click **Cutting Chart** in the Print worktable
2. QuiltCorgi analyzes all patches and identifies those that can be cut from strips:
   - Squares, rectangles, and half-square triangles are ideal for rotary cutting
   - Irregular shapes fall back to template cutting
3. The chart shows:
   - Strip width for each fabric/color
   - Sub-cutting instructions (e.g., "cut 4.5 inch strip, sub-cut into 4.5 inch squares")
   - Total strip length needed

### Combining Methods

Many quilters use rotary cutting for simple shapes and templates for complex ones. QuiltCorgi's cutting chart clearly separates these, so you can use the best method for each patch type.

## Print List

The print list is a comprehensive materials summary. It includes:

- Total yardage per fabric/color (with 10% safety margin)
- Backing and binding yardage calculations
- Thread color recommendations based on your fabric choices
- Batting size recommendation based on finished quilt dimensions

## Export Tips

- Always check the **Preview** before exporting to catch any issues
- Use **Actual Size** printing in your PDF viewer — do not let it scale to fit
- Print a test page and measure a known template to verify accuracy
- For large quilts, consider the **Tiled Print** option that splits oversized templates across multiple pages
